开源之夏 2025 编程活动正在如火如荼的进行中，当前也迎来了报名的倒计时阶段，开源之夏组织方也通过高校行系列活动进入各大高校，帮助高校开发者科普开源文化、开源活动、开源技术。

6 月 4 日 开源之夏携手多位开源技术大咖、经验型选手走进西安电子科技大学，OpenTiny 开源社区也参与开源之夏·西安电子科技大学的高校行活动，本次活动将 OpenTiny 最新演进的开源技术带到课堂，结合当前火热的 AI 话题，与西电学子共同分享与探讨 AI 与前端的关系，并介绍了当前 OpenTiny 社区 15 个技术课题，希望同学们一起参与到开源之夏的活动中来。

## 精彩看点

本次活动一共有 12 个议题，活动现场座无虚席，同学们怀揣着对知识的渴望，早早地来到现场，期待着这场技术盛宴的开启。活动有幸也邀请到了多位行业内的技术专家和资深开发者，他们以深入浅出的方式，围绕开源之夏及各个开源社区的核心技术和应用场景展开了精彩的分享。

活动伊始，西安电子科技大学计算机科学与技术学院党委书记李青山教授登台致辞，为本次活动拉开帷幕。随后，开源之夏活动运营负责人李梦就开源之夏  2025  的各项事宜进行了详尽阐述，涵盖活动历程以及项目价值等多个维度。开源原子基金会教培部顾问张小波也向在场众人介绍了开放原子校源行相关活动，进一步丰富了活动内容。中国科学院软件研究所工程师何若轻以贡献者的视角，分享了融入开源社区的诸多经验与方法。华为技术有限公司开源运营总监马全一深入剖析了  rust  编程语言生态的发展现状与趋势。

蚂蚁集团图计算技术专家戚仕鹏则带来了一场主题为《graph + AI：大模型浪潮下的图计算》的精彩分享，为同学们揭秘前沿技术应用场景。中国科学院软件研究所人事纪然为学生解读了中国科学院研究所的现状及发展前景，助力学子规划未来。高校开发者隐语社区  Contributor  李春阳分享了自己在隐语社区的开源实践经验，为有意参与开源的同学提供了宝贵的参考。MateChat  项目成员刘浩森重点介绍了  MateChat  智能化助手的优势与亮点，展现出该项目的独特魅力。香山开源社区王郅尊分享了《香山：你身边的开源高性能处理器》，阐述了开源硬件的潜力与价值。万众一芯社区负责人王丹宁介绍了《万众一芯：让每个开发者，都能参与  RISC - V  处理器验证》，毕定钧同学还进行了实操演示，让同学们更加直观地了解了万众一芯项目的具体内容与操作流程。

然而在本次活动里，OpenTiny  社区也为高校开发者们精心准备了《基于  MCP  协议：实现  AI  自动操作  Web  组件》的分享。

分享开始后，OpenTiny  开源社区负责人曾令卡先给大家介绍了  OpenTiny  开源社区在开源之夏活动中的项目，欢迎大家踊跃报名参与。随后和大家进行了简单互动，了解大家对前端知识、AI 知识的掌握情况，部分同学纷纷举手回应。而本次分享的重点主要是围绕  OpenTiny  最新的演进成果展开，以全新视角阐释如何赋能  AI  提升前端开发者搭建  Web  应用的效率 。本次议题结合了  MCP  热点话题，深入浅出地讲解了  MCP  工具的概念，以及  TinyVue  智能化组件库如何融合  MCP  协议赋能  AI，实现  AI  操控  Web  组件的原理。同时，通过网页  AI  对话框操作、Copilot  等  MCP Host  操作以及  Coze  等智能体搭建平台操作这三种场景演示，直观呈现其强大功能，最后也呼吁开发者们积极使用并参与开源共建，携手推动  Web  开发领域的创新与发展。

除了精彩的议题分享，整个活动现场还设置了充分的互动抽奖环节。同学们积极参与，共同投入到这场干货满满、福利满满的开源智慧之旅中。活动最后，大家一起进行了合影留念，到此本次活动落下帷幕。![]

随着本次活动的圆满结束，大家都收获了诸多令人欣喜的成果。一方面，激发了同学们对于开源技术的热情和兴趣，学到了实用的技术知识，还开阔了视野，对未来的职业发展方向有了更清晰的认识。另一方面，也让更多开源项目中在高校中被更多开发者所熟知，推动了更多开源技术的发展。同时，本次活动也不是终点，希望有更多开发者一起加入到开源社区的发展与建设中来，实现开源领域的不断创新。
