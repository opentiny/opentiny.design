<h2 id='definition'>TinyTheme 是什么？</h2>

[`TinyTheme`](/designtheme/home) 是一款旨在提高开发效率的主题配置系统。让开发者更专注、让风格更多变。 即支持基于模板主题定制又支持细粒度的视觉需求，为用户打造更好的用户体验。

<h2 id='select'>为什么选择 TinyTheme ?</h2>

华为云开源组件库<a href="/tiny-vue/zh-CN/os-theme/docs/envpreparation" target="_blank" rel="noopener noreferrer">`TinyVue`</a>、<a href="/tiny-ng/overview" target="_blank" rel="noopener noreferrer">`TinyNG`</a>开放了65+基础组件，满足您多场景的需求开发。在此基础上，配合主题配置系统的细粒度所见即所得的风格定制能力，支撑多样化的产品需求。

<h2 id='features'>功能特性</h2>

1. **细粒度的组件编辑**，颗粒度更小的主题配置流程，既保持视觉语言上的统一连贯，又能满足业务和品牌多样化的视觉需求。
2. **便捷主题管理**，多版本多协作的主题管理功能，使得生态更开放。
3. **智能的代码生成**，将用户的配置转换成代码，安装后即可使用。让开发者更专注。
4. **高可用的色盘**，智能色阶算法会为你生成一套更加灵活且高可用的色盘。让风格更多变。
