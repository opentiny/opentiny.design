<h1 id='hwc'>HWC插件</h1>

---
<h2 id='desc'>描述</h2>

> 云服务插件是基于华为云服务 API 提供的命令行管理工具，您可以通过此工具管理和使用您的各类云服务资源

<h2 id='support'>支持的云服务</h2>

- 对象存储服务 OBS

- 虚拟私有云服务 VPC

- 弹性公网 IP 服务 EIP

- 云数据库服务 RDS

- 函数工作流服务 FunctionGraph

- API 网关服务 APIG
